About the Project

I effectively operationalized a powerful NBA sports betting algorithm, achieving an innovative blend of sports analytics and automation. This algorithm, known for its impressive track record of over 50% monthly profit, was originally designed with a sophisticated ensemble of XGBoost and LSTM models to predict money lines and over-unders. Seeing an opportunity to optimize the algorithm's application, I stepped in to automate the entire process. This resulted in eliminating the manual work involved in running the algorithm, significantly increasing its usability and efficiency. My efforts were not just focused on automating the process; I prioritized ensuring the output was easy to comprehend and readily actionable for the end-user.

Features

  • Feature 1: Automated job to run the final optimized algorithm.
  • Feature 2: Simple frontend to grab the most recent output of the algorithm.
  • Feature 3: Email reporting and notifications for the algorithm output (also serves as the "DB").
  • Feature 4: Windows server to support automated daily jobs for running the final algorithm and reporting the live report.

Technologies Used

The project is built using the following technologies:

  • Technology 1: Python is the language used for the algorithm
  • Technology 2: PowerShell is the language used for adminstration of the Windows server
  • Technology 3: HTML is the technology used for the frontend
